#Control flow
-   if
-   elif
-   else

```python
if True:
    print('its true')

if 3 > 2:
    print('its true')

sleeping_hours = 8
if sleeping_hours > 7:
    print('you can drive')
else:
    print('you cannot drive')

num = 4
if num % 2 == 0:
    print('even number')
else:
    print('odd number')
```
